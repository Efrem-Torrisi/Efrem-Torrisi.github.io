Spite: The Depth of Darkness

A Diablo-like action RPG

Role: Technical Artist Team Size: 20 Engine: Custom DX11 Engine (RHE) Year: 2025

My Contributions

This project spanned a 3-month development period at half-time pace (20 hours per week).

Pipeline Tools

Built two pipeline tools to speed up the team's asset workflow. The first is a DDS texture converter with configurable compression settings, allowing artists to batch convert textures into the format the engine expects.

The second is a TGO batch generator that automates the creation of prefab files for the engine, removing the need to manually set them up one by one.

VFX Shaders

Created VFX shaders in HLSL for the game's combat and environmental effects. Two flipbook-based effects were authored in Illugenn and played back using a custom flipbook shader I developed with frame interpolation, allowing smooth playback from fewer source frames and keeping the memory footprint low.

I also built a fire effect done entirely in shader code with a single color gradient texture, and a cooldown shader for abilities in the style of World of Warcraft's radial wipe.

Dithering Camera Shader

Wrote a dithering transparency shader in HLSL to handle camera occlusion. Since the game uses an isometric camera, environment geometry often blocks the player's view.

Instead of hiding or clipping those objects entirely, this shader fades them out with a dithering pattern, keeping the scene readable without breaking the visual continuity.

Rigs & Animations

Rigged and animated three of the game's enemy characters: the Popcorn enemy, the Heavy enemy, and the Boss. Each character has a full animation set including idle, movement, attack, and death sequences.
