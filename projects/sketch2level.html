Sketch2Level

2D Pixel Sketch to 3D Level Assembly Pipeline

Role: Solo Developer Tools: Houdini, Python, Unreal Engine 5 Type: Level Design Pipeline Tool

Overview & Technical Breakdown

A Houdini HDA tool that converts top-down 2D pixel sketches into assembled 3D levels. Each pixel color maps to a modular kit piece (floors, walls, doors, windows, stairs, pillars, corners), and the tool automatically detects wall and corner placement based on floor adjacency.

Supports multi-floor levels by stacking multiple sketch images, and exports levels as a JSON file with coordinate conversion different coordinate systems.

Workflow Demo

Color Legend and Pixel Parsing

Each pixel color maps to a building element type via a fully user-configurable legend in the HDA's parameters. A Python SOP reads the image with PIL, checks each pixel against the legend, and places points at grid positions with the appropriate type attribute.

Floor pixels and special types (stairs, pillars) are placed directly, while wall positions are inferred from floor adjacency in a later pass. The wall pixel's color determines which modular piece is used (standard wall, door, window, etc.).

Wall Placement and Corner Detection

For each floor pixel, the tool checks its four cardinal neighbors and places wall pieces at edges where wall-class pixels are found, with rotation computed automatically.

A second pass detects corner conditions: junctions are classified as inner L-corner, outer L-corner, T-junction, or X-junction based on wall convergence and surrounding floor tiles. Inner corners consume adjacent wall edges to avoid overlap.

Multi-Floor Support

Each floor takes its own PNG via the HDA's multiparm input. The tool processes them sequentially, offsetting each floor vertically based on the modular kit's wall height, stacking simple top-down drawings into a 3D structure.

JSON Export

Once the level is assembled, the HDA can export a JSON manifest containing every placed piece's mesh name, position, rotation, and scale. The export supports coordinate conversion for compatibility with engines that use different coordinate systems.

Unreal Engine Import

A Python script running via an Editor Utility Widget imports the JSON, matching each piece's mesh name against a configurable asset path and spawning Static Mesh actors at the correct positions and rotations. Coordinate conversion is handled by the JSON export, so no additional transform logic is needed.

Since each piece is spawned as its own independent actor, level designers can freely move, delete, or replace any piece after import, using the assembled layout as a starting point for manual iteration.
